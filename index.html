Classes for quick use

Class What it does
Flex classes
d-flex sets the display: flex; element
justify-lg-btw distributes elements along the container axis on a large screen min 1440px
justify-md-btw distributes elements along the container axis on a medium screen min 1024px
justify-sm-btw distributes elements along the container axis on a small screen min 768px
justify-xs-btw distributes elements along the container axis on a extra small screen min 0px
flex-column positions the elements in a column
JavaScript Animations
js-common-up the element rises smoothly from bottom to top
js-common-down the element smoothly descends from top to bottom
js-common-left the element will smoothly come out on the right side
js-common-right the element will smoothly come out of the left side
number-count p behaves like a timer counting from zero to the set value
data-delay="0.2" (NOT A CLAS! DATA ATTIBUTE) responsible for the delay before the animation starts. Works with both js-common-direction and number-count
js-animated-grid wrapper class for four animation blocks
leading-items__item the class of the block that will be animated. The child element of the js-animated-grid class
Options for text
text-center positions the text in the center
